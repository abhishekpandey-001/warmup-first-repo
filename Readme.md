### 🔹 FloCard
- It's a digital networking platform to connect individuals, brands, and organizations.
- It replaces traditional paper visiting cards.
- Supports environmental sustainability by planting a tree for every FloCard created.
- Allows users to:
  - Create multiple cards
  - Share via QR or code
  - Manage connections and communicate

---

### 🔹 Ecosystem Understanding
- **Individuals**: Create and share FloCards, connect, and communicate.
- **Communities**: Collaborate and work together in groups.
- **Tree Planters**: Track tree growth, monitor impact, and manage plantation projects.
